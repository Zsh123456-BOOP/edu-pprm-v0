# Edu-PPRM Related Literature Pack

This folder stores open-access papers and focused reading notes used to evaluate
whether Edu-PPRM-v0 has research value after Phase 3.17.

## Scope

Current narrowed idea:

```text
first_wrong_step -> coarse minimal_repair_type
```

The literature review asks whether this is a meaningful research gap between:

- process supervision / PRM / step error localization;
- educational tutoring feedback and remediation;
- non-leaking, minimal, actionable tutor guidance.

## Folder Layout

- `pdfs/`: downloaded open PDFs from official sources.
- `text/`: extracted text from those PDFs for local reading/search.
- `readers/`: focused nature-reader style source-grounded reading artifacts.
- `metadata/`: download metadata and source URLs.

## Key Decision

The idea is research-relevant if kept narrow:

```text
step-level error localization + pedagogical repair policy
```

It should not currently be framed as a mature contribution around
`earliest_actionable_step`, because Phase 3.17 validated the 6-class repair
taxonomy more strongly than the earliest-actionable distinction.
