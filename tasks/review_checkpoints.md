# Review Checkpoints

## Review Gate 1: 数据源是否足够支撑 pilot

Stop after Phase 1. Do not continue to Phase 2 until the user reviews:

1. all source summary JSON files
2. 20 examples per source
3. field coverage
4. fields that can map to first-error labels and teacher feedback
5. fields that cannot directly map to pedagogical intervention labels

Possible decisions:

- continue Phase 2
- fix source mappings first
- remove low-quality sources
